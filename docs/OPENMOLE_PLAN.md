# 摩尔庄园白盒改造路线（基于 fork Jerenaux/phaserquest）

> 本文档由 Devin 通读 phaserquest 源码后撰写，目的是规划在 `fxq45/phaserquest` 上做
> 「摩尔庄园白盒」式社区养成页游的改造路径。**白盒阶段**：不做美术，全部用纯色方块 +
> 文本标签代指，只验证玩法、网络、状态机。

## 1. phaserquest 现状架构（实读代码后摘要）

| 维度 | 实现 | 评价 |
|---|---|---|
| **客户端引擎** | Phaser **2.x** (CE 版，无 TS) | 老但稳，2D 像素 RPG 最合适 |
| **服务端** | Node.js + Express + socket.io **1.7** | 老，得升 |
| **同步协议** | 自定义二进制（CoDec.js / Encoder / Decoder） | 写得很精巧，省流量；但对白盒阶段是过早优化，建议白盒先关二进制走 JSON |
| **AOI（视野裁剪）** | 自己实现的网格 AOI（每格 34×20 tile，共 6×16=96 格） | **白盒阶段直接复用，省你三周** |
| **地图** | Tiled `.tmx` + 客户端 / 服务端两份 `.json` | 后面摩尔庄园白盒可以把整张地图换成纯色 tile |
| **存储** | MongoDB（玩家 doc：位置、装备、生命值） | 直接换 schema 改成摩尔庄园的「摩尔豆 / 家具 / 拉姆」即可 |
| **寻路** | `pathfinding` 库 | 复用 |
| **战斗** | Monster + Item + 自定义伤害循环 | **摩尔庄园不需要，删除** |

代码体量：服务端 ~2000 行 JS，客户端 ~3000 行 JS，**两天就能读完**。

## 3. 白盒里程碑（建议节奏）

### M0：环境跑通（0.5 天）
- fork → 本地能用 docker-compose 启服务端 + MongoDB
- 客户端浏览器能登录、看到 phaserquest 原版的玩家小人在跑
- **交付**：原版能跑

### M1：拆战斗系统 / 仅保留社交骨架（1 天）
- 把 Monster / 战斗 / 装备 / 死亡复活相关全部停掉（标记 deprecated，不删源码以便参考）
- 玩家只剩：移动、聊天、AOI 同步
- 地图先用原版的（暂不换白盒）
- **交付**：可以多人在原版地图里聊天 + 走动

### M2：白盒地图替换 + 区域命名（1 天）
- 用 Tiled 重画一张 16×16 单色 tile 的地图，划出 4 个区域：
  - `爱心广场`（白色方块地）
  - `拉姆农场`（绿色方块地）
  - `摩尔城堡入口`（灰色方块地）
  - `淘淘乐街`（黄色方块地）
- 用 Phaser BitmapText 在地面上画 `[爱心广场]` 等大字
- **交付**：白盒地图能跑

### M3：Realm（场景切换）系统（1.5 天）
- 抽象 `Realm` = 一组独立的 AOI + 玩家列表
- 加传送阵碰撞体：玩家走进 → 切换 realm
- 加私人家园 realm：每玩家一个独立 realm
- **交付**：能切场景了

### M4：背包 + 摩尔豆 + 家具（1 天）
- 改 Mongo schema：`{moerDou: 0, furniture: [], lamu: null}`
- 复用 phaserquest 的 Item Pickup → 改成「家具掉落」
- 客户端用半透明灰底 UI 框列家具
- **交付**：能在家园放白盒家具

### M5：第一个小游戏（泡泡龙白盒）（2 天）
- 走进小游戏触发点 → 弹出小游戏 realm（独立 Phaser scene）
- 用纯几何形状画白盒泡泡龙
- 结算后给玩家加 `moerDou`
- **交付**：跑通"主世界 ↔ 小游戏"循环

### M6：拉姆 + SMC 任务（2 天）
- 给玩家加一只白盒方块拉姆跟在身边
- 任务系统：NPC 触发 → 完成 → 奖励
- **交付**：完整养成循环

**总计：约 9 天白盒 MVP**，前提是你每天能投 4-6 小时。

## 5. 风险提示

1. **Phaser 2 已停止主线维护**（CE 版社区在维护）。白盒阶段没关系，正式开发前要决定升 Phaser 3。
2. **MongoDB 3.6 SDK 已 EOL**。本地开发没问题，部署到云数据库时要升级。
3. **摩尔庄园是注册商标**。仓库名和对外宣传都不能直接叫"摩尔庄园"。建议项目代号 `OpenMole` 或 `BoxedManor`。
4. **phaserquest 用了 Tiled v1.1.6 之前的 JSON 格式**，更高版本 Tiled 导出会不兼容（README 已警告）。白盒阶段建议直接用 1.1.6。
